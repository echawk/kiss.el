* kiss.el

This is an implementation of the kiss package manager in Emacs lisp.

https://github.com/kiss-community/kiss

Currently is very, very much WIP.

** Features Implemented:

- [ ] alternatives
- [ ] build
- [ ] checksum
- [ ] download
- [ ] install
- [X] list
- [ ] remove
- [X] search
- [ ] update
- [ ] upgrade
- [ ] version

Feel free to submit issues & PRs!
